## Task Manager CLI (Node.js)
A simple command-line tool to manage tasks stored in `tasks.json`.

### Install & Run
- Requires Node.js and npm.

### Commands
- `node src/index.js help` — show help
- `node src/index.js add "Task title"` — add a task
